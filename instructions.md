# GitHub Copilot Instructions

This project was developed with GitHub Copilot.

## Coding Guidelines

- Use Python 3.10+
- Follow PEP 8
- Keep code modular and reusable
- Preserve existing Sudoku functionality while refactoring
- Separate Flask routes from Sudoku logic
- Use type hints where appropriate
- Handle exceptions properly
- Generate Sudoku puzzles with exactly one unique solution
- Use pytest for testing
- Keep HTML semantic and accessible
- Make the UI responsive
- Support Dark Mode
- Store leaderboard data using browser localStorage

## Testing

Run:

```bash
python -m pytest
```

All tests should pass before committing changes.
